# Disaster Risk Intelligence — Workspace

Dashboard web real-time gempa bumi Indonesia (portofolio project).

## Struktur

```
disaster-risk-intelligence/    ← workspace root (tanpa git)
├── backend/                   ← git repo Go service
└── frontend/                  ← git repo Next.js app
```

Dua repo Git terpisah. Untuk kerja di salah satu, buka terminal di folder tersebut.

## Spec

Lihat `disaster-risk-intelligence-spec (1).md` di root folder ini untuk detail lengkap arsitektur, data sources, dan rencana kerja.
